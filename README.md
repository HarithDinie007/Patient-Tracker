# Patient-Tracker
Requires:  
TinyCircuit  
TinyScreen  
TinyShield WiFi Module  

Setup:  
1. Activate hotspot on host device
2. Ensure that SSID is set to 'PatientTracker' and Password is set to 'P@ssword123' (CASE-SENSITIVE and no quotation marks)
3. Upload the PatientTracker.ino into the TinyCircuit
4. Use preferred IDE to run the main.py on the host device
5. Follow the instructions on the TinyScreen to calibrate
